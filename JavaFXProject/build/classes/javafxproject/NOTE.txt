- Download JavaFX from: https://gluonhq.com/products/javafx/
	- Get the Windows architecture, 64 bit, SDK file
- Extract the downloaded file to 'C:/Program Files/Java'
- Open NetBeans: 
	- Click on Tools > Libraries
	- Click New Library
	- Type JavaFX and click OK
	- Click on 'Add JAR/Folder' and go-to 'C:/Program Files/Java/[JavaFX_Folder_Name]/lib'
	- Select all the files inside it and click OK
	- Right click the JAVA Project > Properties and go-to Libraries section, Add Libraries, and then select JavaFX and then click Add Library
		- On same panel, Go-to Build > Compiling and uncheck 'Compile on Save'
		- On same panel, Go-to Run, and Paste the script inside 'VM Options' textarea and click OK: 
			--module-path "C:\Program Files\Java\javafx-sdk-23\lib" --add-modules javafx.controls,javafx.fxml,javafx.base,javafx.graphics

- To run the app, right click project, select 'Clean and Build'
	- Then run the app from command prompt as: 
		- java -jar "C:\Users\Nagarjuna 14\Documents\Java-Programming\JavaFXProject\dist\JavaFXProject.jar"
